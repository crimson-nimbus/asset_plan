## 2. 各リスク要因の詳細分析

### 2.1 信用リスク（违约风险）

**定義**: 発行体や取引相手が約束した利払い・償還を行えなくなるリスク

| 商品 | リスク水準 | 説明 |
|---|---|---|
| 銀行預金 | 極低 | 預金保険50万元保証。**複数行に分散すれば実質元本保証** |
| 国債 | ほぼゼロ | 中国政府の信用力。デフォルト歴なし |
| 養老理財 | 低 | 銀行理財子会社が運用。R2等級だが元本保証ではない |
| 公募REITs | 低〜中 | 裏付け資産の収益力に依存。特許経営権は期限あり |
| 紅利ETF | 低〜中 | 個別銘柄リスクはETFで分散。指数連動 |

**対策**:
- 銀行預金は1行あたり50万元以下に分散
- 養老理財はR2（中低リスク）以下の商品のみ選択
- REITsは裏付け資産の質が高い（高速道路・保障住宅）ものを優先

### 2.2 金利リスク（利率风险）

**定義**: 市場金利の変動により、保有資産の価値や将来の収益が変化するリスク

**中国の金利環境（2025〜2026年）**:
- 中国人民銀行は緩和的金融政策を継続
- 10年国債利回りは「1%時代」に突入（1.3%〜2.0%）
- 銀行預金金利は継続的に低下トレンド

**影響**:
- 定期預金・国債: 満期後の再投資時に低い利率で再設定されるリスク（再投資リスク）
- 養老理財: 運用資産の利回り低下による分配金減少
- REITs: 金利低下局面ではREITsの相対的な魅力度が上昇（価格上昇）

**対策**:
- ラダー戦略（1年・3年・5年に分散）で金利リスクを平滑化
- 現在の利率で長期（5年）国債・大額存単を多めに確保
- 金利低下局面では変動金利商品（養老理財）が相対的に有利

### 2.3 流動性リスク（流动性风险）

**定義**: 必要な時に資金を引き出せない、または不利な条件でしか換金できないリスク

| 商品 | 換金性 | 換金時の損失 |
|---|---|---|
| 銀行預金 | 即日可能 | 中途解約は活期金利適用（大幅減） |
| 大額存単 | 転譲可能な場合あり | 転譲市場の厚みに依存 |
| 国債 | 中途換金可能 | 利息から手数料控除 |
| 養老理財 | **5年間換金不可** | 封鎖期中は原則引出不可 |
| 養老保険 | 中途解約可能 | **大幅な元本割れ** |
| REITs | 取引所で売却 | 市場価格での売却、NAV割れリスク |
| 紅利ETF | 取引所で即売却 | 市場価格での売却 |
| 黄金ETF | 取引所で即売却 | 市場価格での売却 |

**最大の注意点**: **養老理財の5年封鎖期**。60歳で購入すると65歳まで引き出せない。突発的な医療費等に対応する**流動性バッファ**を別途確保する必要がある。

**対策**:
- 生活費6〜12ヶ月分を普通預金/活期に常時保持
- 養老理財への配分は全資産の25%以下に抑える
- ETF・REITs部分が「いざという時の換金層」として機能

### 2.4 市場リスク（市场风险）

**定義**: 市場全体の下落により資産価値が減少するリスク

**中国株式市場の特徴**:
- A株市場は変動が大きい（2015年株式ショック: −40%超）
- 個人投資家の比率が高く、投機的な動きが出やすい
- 政策主導で急変動する場合がある

**各商品への影響**:

| 商品 | 最大想定損失（年間） | 過去の最大ドローダウン |
|---|---|---|
| 紅利ETF | −15%〜−25% | 2015年: 約−30% |
| 公募REITs | −10%〜−20% | 2022〜2023年: 約−20% |
| 黄金ETF | −10%〜−20% | 短期的に−15%程度 |
| 養老理財 | −2%〜−5% | 2022年理財破净潮: 一時−3%程度 |
| 銀行預金・国債 | 0% | なし（元本保証） |

**対策**:
- 紅利ETFは定時定額（ドルコスト平均法）で段階的に購入
- 市場暴落時に慌てて売却しない心構え（長期保有を前提）
- REITs・ETF合計が全資産の40%を超えないようにする

### 2.5 政策リスク（政策风险）

**定義**: 政府の政策変更により投資環境が変化するリスク

**中国特有の政策リスク**:
- **保険予定利率の引き下げ**: 3.5% → 3.0% → 2.5% → 2.0%と急速に低下
- **銀行理財の「净值化」**: 予想収益率型から净値型（時価評価）への移行で、一時的に「破净」（元本割れ）が発生
- **REITs制度の変更**: 税制・配当ルール・対象資産範囲の変更
- **資本規制**: 送金制限、外貨両替制限の変更
- **社会保障制度改革**: 退職年齢引き上げ、養老金支給額の変動

**対策**:
- 1つの政策に依存しすぎない分散配置
- 制度変更のニュースを定期的にフォロー
- 規制変更があっても慌てず、長期的視点で対応

## 3. 60歳以上特有のリスク要因

### 3.1 長寿リスク（长寿风险）

予想以上に長生きすることで、資産が尽きるリスク。

- 中国の平均寿命: 約78歳（2025年時点）
- 計画上は**90歳まで**のキャッシュフローを想定すべき
- 元本を取り崩すスピードが速すぎないよう、**利子・配当収入で生活費を賄う**設計が重要

### 3.2 医療費リスク（医疗费用风险）

- 中国の医療保険制度（医保）はカバー範囲が限定的
- 重大疾病の自己負担が数十万元に達する場合がある
- **対策**: 医療保障の確認、重疾険（大病保険）の検討、緊急用流動資金の確保

### 3.3 インフレリスク（通胀风险）

- 中国のCPI: 2〜3%（変動あり）
- 定期預金利率1.25%ではインフレに負ける
- **実質リターン** = 名目利回り − インフレ率
- パターンAでは実質リターンがマイナスになりうる

### 3.4 認知能力低下リスク

- 高齢化に伴い、投資判断能力が低下する可能性
- 詐欺・不適切な金融商品の被害リスク
- **対策**:
  - 資産構成は**できるだけシンプル**に保つ
  - 家族（本件ではお子さん）が定期的にチェック
  - 70歳以降は能動的な投資判断が必要な商品（REITs・ETF）の比率を下げる
  - 銀行預金・国債等の「自動運転」可能な資産を中心に

### 3.5 詐欺リスク（诈骗风险）

- 中国では高齢者を狙った金融詐欺が多発
- 「高利回り保証」の理財商品は**ほぼ確実に詐欺**
- P2P理財の崩壊（2018〜2020年）で多くの高齢者が被害
- **対策**:
  - **正規の金融機関のみ利用**（国有大手銀行、証監会認可の証券会社）
  - 年利5%超を「保証」する商品には手を出さない
  - 民間の理財会社、知人の紹介による投資には一切応じない

## 4. ストレステスト

### シナリオ1: 銀行金利さらに低下（LPR引下げ）

| 影響 | パターンA | パターンB | パターンC |
|---|---|---|---|
| 預金・国債の利回り | −0.3% | −0.1% | −0.1% |
| 養老理財 | 影響小 | −0.2% | −0.2% |
| REITs（相対的魅力度UP） | なし | +0.1% | +0.2% |
| **全体影響** | **−0.3%** | **−0.2%** | **−0.1%** |

### シナリオ2: 株式市場暴落（−30%）

| 影響 | パターンA | パターンB | パターンC |
|---|---|---|---|
| 紅利ETF | なし | −4.5% | −7.5% |
| REITs（連動下落−15%） | なし | −2.25% | −3.0% |
| その他 | 影響なし | 影響なし | 影響なし |
| **全体影響** | **なし** | **−6.75%** | **−10.5%** |

### シナリオ3: インフレ加速（CPI 5%）

| 影響 | パターンA | パターンB | パターンC |
|---|---|---|---|
| 実質リターン | **−3.0%** | **−1.5%** | **−0.5%** |
| 黄金（インフレに連動上昇） | +5% | +5% | +5% |
| 実質リターン（黄金込み） | **−2.5%** | **−1.0%** | **0%** |

> 💡 **結論**: パターンBは最も「あらゆるシナリオに対してバランスが取れている」。パターンAはインフレシナリオに弱く、パターンCは株式暴落シナリオに弱い。

## 5. リスク軽減戦略まとめ

| 戦略 | 対象リスク | 具体的アクション |
|---|---|---|
| **分散配置** | 信用・市場 | 銀行は複数行に分散、商品カテゴリも分散 |
| **ラダー戦略** | 金利・流動性 | 預金・国債の満期を1/2/3/5年に分散 |
| **流動性バッファ** | 流動性 | 生活費6〜12ヶ月分を普通預金に維持 |
| **段階的購入** | 市場 | REITs・ETF・黄金はドルコスト平均法で購入 |
| **年齢連動リバランス** | 全般 | 5年ごとに安全資産比率を5%〜10%引き上げ |
| **定期チェック** | 認知・詐欺 | 家族が年1〜2回ポートフォリオを確認 |
| **正規機関のみ** | 詐欺 | 国有大手銀行・認可証券会社のみ利用 |
