# 📈 推奨ポートフォリオ設計

> 中国在住60歳前後の退職者向け / 安定収入＋資産保全

---

## 設計思想

### 核心原則

1. **元本保全を最優先** — 60歳以上は損失回復の時間が限られる
2. **定期的なキャッシュフロー確保** — 年金＋配当・利子で生活費を賄う
3. **インフレに負けない** — CPI 2〜3%を上回る実質リターン
4. **流動性のバランス** — 突発的な医療費等に対応可能な資金を確保
5. **段階的な構築** — 一括投入ではなく時間分散で構築

### 資産配分のフレームワーク

```
┌────────────────────────────────────────┐
│          防衛資金（生活費6〜12ヶ月分）        │  ← まず確保
├────────────────────────────────────────┤
│     安全資産（預金・国債）: 30%〜55%         │  ← 核心
├────────────────────────────────────────┤
│     安定収益資産（養老理財・REITs）: 20%〜40%  │  ← 収益強化
├────────────────────────────────────────┤
│     成長資産（紅利ETF）: 0%〜25%            │  ← 適度な成長
├────────────────────────────────────────┤
│     ヘッジ資産（黄金）: 5%〜10%              │  ← リスクヘッジ
└────────────────────────────────────────┘
```

## パターンA: 超保守型

> **目標年化利回り: 2.0%〜2.5%** / リスク許容度: 極めて低い

### 対象者

- リスクを一切取りたくない方
- 資産の安全性を最大限に重視する方
- 元本割れが絶対に許容できない方
- 高齢（70歳以上）の方

### 配分

| 商品カテゴリ | 配分 | 期待利回り | 寄与度 |
|---|---|---|---|
| 銀行定期預金・大額存単 | **40%** | 1.3%〜1.55% | 0.52%〜0.62% |
| 国債（储蓄国债） | **30%** | 1.8%〜2.0% | 0.54%〜0.60% |
| 養老保険（安定型口座） | **20%** | 2.5%〜3.5% | 0.50%〜0.70% |
| 黄金ETF | **10%** | 0%（保値目的） | 0% |
| **合計** | **100%** | | **1.56%〜1.92%** |

※ 黄金の価格上昇を加味すれば2.0%〜2.5%は達成可能

### 具体的な配置例（100万元の場合）

| 商品 | 金額 | 具体的配置 |
|---|---|---|
| 銀行預金 | 20万元 | 工商銀行3年定期 × 1、建設銀行3年定期 × 1（各銀行50万以下） |
| 大額存単 | 20万元 | 農業銀行3年大額存単 |
| 国債 | 30万元 | 储蓄国債（电子式）3年・5年各15万元 |
| 養老保険 | 20万元 | 専属商業養老保険・安定型口座 |
| 黄金ETF | 10万元 | 黄金ETF（518880）段階的購入 |

### 年間予想収入

- 年間利息・収益: 約15,600元〜19,200元
- 月平均: 約1,300元〜1,600元

## パターンB: 安定成長型（推奨）

- 元本の安全性を重視しつつ、インフレに対抗したい方

## 3パターン比較

| 指標 | パターンA<br>超保守型 | パターンB<br>安定成長型 | パターンC<br>適度積極型 |
|---|---|---|---|
| 目標利回り | 2.0%〜2.5% | 3.0%〜3.5% | 3.5%〜4.5% |
| 月間収入（100万元） | 1,300〜1,600元 | 2,300〜2,842元 | 2,567〜3,233元 |
| 元本安全性 | ★★★★★ | ★★★★☆ | ★★★☆☆ |
| インフレ耐性 | ★★☆☆☆ | ★★★★☆ | ★★★★★ |
| 流動性 | ★★★★☆ | ★★★★☆ | ★★★★☆ |
| 適切な年齢 | 70歳以上 | **60〜70歳** | 60〜65歳 |
| 最大予想損失 | ≈0%（黄金除く） | ≈−5%〜−8% | ≈−8%〜−12% |

> 💡 **推奨**: 多くの60歳前後の退職者には**パターンB（安定成長型）**が最適。年齢を重ねるにつれて、パターンA寄りに調整（紅利ETF・REITsの比率を下げ、預金・国債の比率を上げる）するのが理想的。
