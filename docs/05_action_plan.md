# 🎯 実行アクションプラン

> ステップバイステップの行動計画 / 3つのフェーズで段階的に実行

---

## 概要

資産運用計画を「一気に始める」のではなく、**3つのフェーズに分けて**段階的に実行する。これにより、市場タイミングリスクを分散し、各ステップで学びながら進められる。

```
Phase 1（即座〜2週間）: 基盤整備 + 安全資産の配置
        ↓
Phase 2（1〜3ヶ月）: 養老理財・保険商品の選定・契約
        ↓
Phase 3（3〜6ヶ月）: 成長資産・ヘッジ資産の段階的購入
```

---

## Phase 1: 基盤整備 + 安全資産の配置（即座〜2週間）

### Step 1.1: 現状の棚卸し ✅まず最初に

- [ ] 現在の全資産を一覧にまとめる（銀行預金、保険、不動産 等）
- [ ] 月間の収入（年金、その他）と支出を把握する
- [ ] 既に加入している保険・理財商品を確認する
- [ ] 借入金・負債がないことを確認する
- [ ] **生活防衛資金**を計算する（月間支出 × 6〜12ヶ月）

> 📝 **テンプレート**:
> | 資産種別 | 金融機関 | 金額 | 利率/利回り | 満期日 |
> |---|---|---|---|---|
> | 普通預金 | ○○銀行 | ○○万元 | 0.05% | なし |
> | 定期預金 | ○○銀行 | ○○万元 | ○○% | 20XX年X月 |
> | ... | ... | ... | ... | ... |

### Step 1.2: 預金の最適化

- [ ] 生活防衛資金（月支出×6ヶ月分）を普通預金に確保
- [ ] 残りの預金を**複数銀行**に分散（各銀行50万元以下）
- [ ] 大額存単の発売情報を確認（候補銀行: 工商銀行、建設銀行、農業銀行）
- [ ] 阶梯式（ラダー）配置の実行:
  - 1/3 → 1年定期 or 大額存単
  - 1/3 → 3年定期 or 大額存単
  - 1/3 → 国債購入用に普通預金に待機

### Step 1.3: 国債の購入準備

- [ ] ネットバンキングの開設・確認（国債購入に必要）
- [ ] 储蓄国債の発行スケジュールを確認（財政部公式サイト）
- [ ] 発行日に合わせて購入実行（**発行当日午前中に操作**）
- [ ] 目標: 电子式储蓄国債（3年 + 5年）を合計で目標配分額まで購入

> ⚠️ **储蓄国債は毎月10日前後に発行、即日完売が多い。事前にネットバンキングの操作を練習しておくこと。**

## Phase 2: 養老理財・保険商品の選定（1〜3ヶ月）

### Step 2.1: 養老理財商品の調査・購入

- [ ] 以下の銀行で養老理財商品のラインナップを確認:
  - 工銀理財（工商銀行系）
  - 建信理財（建設銀行系）
  - 光大理財（光大銀行系）
  - 招銀理財（招商銀行系）
  - 中銀理財（中国銀行系）
- [ ] 確認ポイント:
  - リスク等級: **R2（中低リスク）以下** のみ
  - 封鎖期: 5年が主流だが短いものがあれば優先
  - 過去の年化収益率: 3%以上を目安
  - 起購金額: 1元〜（多くは低額から購入可能）
- [ ] 選定した商品を目標配分額まで購入

### Step 2.2: 商業養老保険の検討（任意）

- [ ] **全体の配分で保険が必要な場合のみ**実行
- [ ] 候補商品の調査（専属商業養老保険の安定型口座）
- [ ] 確認ポイント:
  - 保険会社の格付け（偿付能力充足率）
  - 安定型口座の過去の結算利率
  - 退保（解約）時の现金价值（元本割れの程度）
  - 受取開始年齢と受取方法
- [ ] ⚠️ **60歳からの新規加入は保険料が高額。メリットが限定的な場合は見送り**

### Step 2.3: 個人養老金口座の開設（該当する場合）

- [ ] まだ退職していない場合のみ実行
- [ ] 指定銀行で個人養老金口座を開設
- [ ] 年間12,000元の範囲内で養老金理財商品に投資
- [ ] ⚠️ すでに退職済みの場合はスキップ

## Phase 3: 成長・ヘッジ資産の段階的購入（3〜6ヶ月）

### Step 3.1: 証券口座の開設

- [ ] 証券会社で**A股口座**を開設（REITs・ETF取引に必要）
- [ ] 推奨証券会社: 中信証券、華泰証券、国泰君安、招商証券 等の大手
- [ ] 手数料: 万分之2.5以下を交渉（ネット証券は安い傾向）
- [ ] ⚠️ **ご両親が操作に不慣れな場合は、家族が代わりに操作方法を教える or 付き添う**

### Step 3.2: 公募REITsの購入

- [ ] 以下の優先順で調査・購入:
  1. **高速道路REIT**（分配利回り5.5%〜9.0%、最優先）
  2. **保障性賃貸住宅REIT**（分配利回り3.98%〜4.47%）
  3. **エネルギーインフラREIT**（安定したキャッシュフロー）
- [ ] 銘柄選定のチェックリスト:
  - [ ] 裏付け資産の質と所在地
  - [ ] 過去の実際の分配金額と利回り
  - [ ] NAVとの乖離率（大幅プレミアムは避ける）
  - [ ] 運営管理者の信頼性
- [ ] **一括購入ではなく、3〜6回に分けて段階的に購入**

### Step 3.3: 紅利ETFの購入

- [ ] 候補ETF:
  - 華泰柏瑞上証紅利ETF（510880）
  - 中証紅利ETF（515080）
  - 深証紅利ETF（159905）
- [ ] **定時定額（ドルコスト平均法）で購入**:
  - 毎月の投資額 = 紅利ETF目標配分額 ÷ 6ヶ月
  - 例: 15万元の配分なら、毎月2.5万元 × 6ヶ月
- [ ] 配当再投資はせず、配当金は受け取る設定に

### Step 3.4: 黄金の購入

- [ ] 推奨: **黄金ETF（518880 等）**で購入（保管不要、売買容易）
- [ ] 代替: 銀行の黄金積立（积存金）を毎月少額で実行
- [ ] **一括購入は避け、3〜6ヶ月かけて段階的に購入**
- [ ] 目標: 全体の10%

## 購入後の管理

### 定期チェックリスト（年2回 / 6月と12月）

- [ ] 全資産残高の確認と一覧の更新
- [ ] 各商品の利回り・分配金の確認
- [ ] ポートフォリオ全体の配分比率の確認
- [ ] 目標配分からの乖離チェック（±5%以上ならリバランス検討）
- [ ] 新たな商品・制度変更のニュースチェック
- [ ] 不審な連絡・営業がなかったかの確認

### リバランスのタイミング

| トリガー | アクション |
|---|---|
| 年齢が5歳上がったとき | 安全資産比率を5〜10%引き上げ |
| 配分乖離が±5%超 | 元の配分に近づけるよう売買 |
| 市場暴落時 | **売却しない**。余資金があれば追加購入を検討 |
| 市場急騰時 | 成長資産の一部を安全資産にシフト |

### 緊急時の対応

| 状況 | 対応 |
|---|---|
| 急な出費（20万元以内） | 普通預金→定期預金の中途解約→紅利ETF売却の順で対応 |
| 大きな出費（20万元超） | 上記＋REITs売却で対応。養老理財は封鎖期中は引出不可 |
| 健康問題 | まず医保で対応、不足分は流動資産から |

```
           ━━━━━━━━━━━━━━━━━━━━━━━━━━

Month 1-3: [Phase 2] 養老理財商品の調査・購入 → 保険検討
           ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Month 3-6: [Phase 3] 証券口座開設 → REITs購入 → ETF購入 → 黄金購入
           ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Month 7+:  [運用開始] 定期チェック → 年齢連動リバランス
           ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## ⚠️ 絶対にやってはいけないこと

1. ❌ **「元本保証」「高利回り」を謳う民間理財会社への投資**
2. ❌ **知人・親戚の紹介による投資話への参加**
3. ❌ **P2P、信托（信用が疑わしいもの）、仮想通貨への投資**
4. ❌ **一括で全資産を1つの商品に投入**
5. ❌ **株式の個別銘柄への投資（ETFのみに限定）**
6. ❌ **レバレッジ（借入）を使った投資**
7. ❌ **生活防衛資金を投資に回す**
8. ❌ **よく理解していない商品への投資（仕組みを理解してから購入）**
