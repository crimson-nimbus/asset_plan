# 📊 资产配置模拟器 / 資産配分シミュレーター

中国国内で投資可能な8種の資産クラスを対象とした、年齢別資産配分シミュレーションツール。

全ての計算はブラウザ内で完結し、サーバーへのデータ送信は一切ありません。

## 特徴

- 🎯 **年齢別推奨** — 22〜75歳対応、5段階の投資スタイルプリセット
- 📊 **キャッシュフローシミュレーション** — 年金収入を含む年度別明細
- 🔍 **投資キーワードガイド** — 各資産クラスの検索方法を案内
- 🌐 **バイリンガル** — 中国語 / 日本語 切替対応
- 🔒 **プライバシー保護** — 純粋な静的ページ、データ送信なし
- ⚖️ **法的免責事項** — 中国証券法 / 日本金融商品取引法 準拠
